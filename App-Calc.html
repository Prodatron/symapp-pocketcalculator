1000-Introduction


INTRODUCTION TO POCKET CALCULATOR

Pocket Calculator is a small and useful application for basic and scientific calculations.

1100-Feature overview


FEATURES

The pocket calculator was the first application available for SymbOS and has been rewritten after two years.

1200-What's new?


WHAT'S NEW?

V1.0 (2006-11-21)

- Re-implementation of the floating point routines by using the library from the Amstrad CPC Locomotive Basic
- new scientific mode
V0.1 (2004-09-26)

- The Pocket Calculator is the first running SymbOS application ever. The first version has a buggy floating point implementation.
